⚙ ابزار های کاربردی : 
    📎لینک نیم بها: 
        !nim link 
        به جای link لینکتون رو بگذارید حتما http یا https داشته باشه. 
    
    🔍 جستجو : 
        !search [موضوع] 
       (مثل !search [اسب]) این را بفرستید و به جای موضوع موضوع خود را بگذارید) 
 
    🔍 جستجو کامل : 
        !search-k [موضوع] 
       (مثل !search-k [آهو]) این را بفرستید و به جای موضوع موضوع خود را بگذارید 

    📸 جستجو با عکس : 
        !search-i [موضوع]
        به جای موضوع موضوعتون رو بفرستید
(مثل !search-i [ایران] از این مورد کمتر استفاده کنید چون ربا تز طرف روبیکا محدود است)

    💵 نرخ ارز : 
        !currency را بفرستید 

    💰 نرخ طلا : 
        !gold را بفرستید 

    👤 گرفتن اطلاعات یک کاربر : 
        !info @user 
         (مثل !info @aliyazdani04 ) این را بفرستید به جای user آیدی اون کاربر رو بگذارید 
 
    ℹ 100 فونت شاخ انگلیسی : 
        !font [text] 
       (مثل !font [ali] ) به جای text متنتون نتیجه با 100 فونت ارسال میشه 

    ℹ فونت شاخ فارسی : 
        !font-fa [text]
       (مثل !font-fa [علی] ) به جای text متنتون'

    📋 سرویس ترجمه : 
        !trans [lang:text] 
          (مثل !trans [en:سلام] ) این را بفرستید به جای lang زبان مورد نظر مثلا en انگلیسی به جای text متن مورد نظر 
       

    🗣سرویس متن به گفتار (انگلیسی) : 
        روی پیام خود 
        !speak را ریپلای کنید 

    🔅 آب وهوا : 
        !weather [city]
        بفرستید به جای city شهرتون رو بگذارید 

    📡 پینگ سایت : !ping [site] 
        به جای site سایت مورد نظر بدون http و https و www 

    🖨 اطلاعات یک دامنه : !whois [site] 
        اینم بدون http و https و www به جای سایت سایتتون 

    ℹ گرفتن اطلاعات آیپی : 
        !ip [your_ip] 
        به جای your_ip آیپی تون رو بگذارید 

    🆎 معنی و مفهوم کلمه : !vaj [کلمه ] 
        به جای کلمه کلمتون رو بگذارید فقط کلمات فارسی پشتیبانی میشه 
    
    📃جستجو در مقاله های ویکی پدیا : 
        !wiki-s [title]
       (مثل !wiki-s [اسب] ) به جای title موضوعتون رو بنویسید تمام مقاله های مرتبط براتون لیست میشه 
    
    📑آوردن متن مقاله از ویکی پدیا : 
        !wiki [page:name] 
        به جای page صفحه چندم مقاله رو بزارید مثلا 1 یعنی صفحه اول و به جای name موضوع مقالتون و بعد بفرستید اگر اسم دقیق موضوع مقالتون رو نمیدونید از دستور بعدی جستجو اش کنید
( مثل !wiki [1:روبیکا] )

    🆚 عکس از متن در ویژوال استودیو : 
        روی پیام خود 
        !shot را ریپلای کنید 
        
     یه پیام را چند بار نفرستید و اگر عکس و ویس نیومد براتون محدودیت از طرفه روبیکاس
